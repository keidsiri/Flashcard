Javascript and JQuery Terms

JavaScript(JS)

- an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

- capable of manipulating a certain value or operand. Operators are used to perform specific mathematical and logical computations on operands.

Variables

- anything that can vary. In JavaScript, a variable stores the data value that can be changed later on.

Variable naming conventions

- style guidelines that typically cover naming and declaration rules for variable and functions. Rules for the use of white space, indentation, and comments and programming practices

Functions

- a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it)..

Methods

- the actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

- an object accessible inside functions that contains the values of the arguments passed to that function.

Parameters

- a named variable passed into a function. Function parameters are the names listed in the function's definition.

Return

- a statement is used to return a particular value from the function to the function caller. The function will stop executing when the return statement is called. The return statement should be the last statement in a function because the code after the return statement will be unreachable.

Chaining methods

- also known as Cascading, means repeatedly calling one method after another on an object, in one continuous line of code.

Strings

- The String object is used to represent and manipulate a sequence of characters.

Booleans

- A JavaScript Boolean represents one of two values: true or false.

Undefined

- The global undefined property represents the primitive value undefined

NaN

- The global NaN property is a value representing Not-A-Number.

Null

- The value null represents the intentional absence of any object value

Alert

- The alert() method displays an alert box with a specified message and an OK button.

Comments

- way to deliver message. It is used to add information about the code, warnings or suggestions so that end user can easily interpret the code.

jQuery

- jQuery is a JavaScript library that allows web developers to add extra functionality to their websites.

Attributes

- a collection of objects that belong to a built-in Attr class, with name and value properties
